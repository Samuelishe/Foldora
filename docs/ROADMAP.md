# Roadmap

## MVP 0.1

1. Применить иконку к выбранной папке.
2. Убрать иконку с выбранной папки.
3. Создать новую папку с выбранной иконкой.
4. Хранить 3-5 встроенных стилей.
5. Зарегистрировать legacy context menu в HKCU.
6. Удалить legacy context menu.
7. Открыть простое WPF-окно настроек.
8. Хранить настройки в AppData.

## Не делать в MVP

- Полноценную Windows 11 modern context menu integration.
- COM shell extension.
- Фоновый сервис.
- Автозапуск.
- Marketplace паков.
- Синхронизацию.
- Сложный редактор иконок.
- Патчинг системных DLL.
- Глобальную замену стандартной иконки всех папок Windows.
- Зависимость от сторонних shell-кастомайзеров.

## Будущее

Modern context menu, installer, pack import/export, preview UI и расширенная валидация паков рассматриваются после рабочего MVP.
