# Документация Foldora

Рекомендуемый порядок чтения:

1. `PROJECT_STATE.md` - текущее состояние и фокус.
2. `ROADMAP.md` - ближайший MVP и будущие версии.
3. `ARCHITECTURE.md` - проекты, слои и зависимости.
4. `REQUIREMENTS.md` - платформа и ограничения.
5. `DESKTOP_INI.md` и `SHELL_INTEGRATION.md` - ключевые Windows-механизмы.
6. `PACKS.md`, `SETTINGS.md`, `CLI.md`, `UI_DESIGN.md` - пользовательские форматы и интерфейсы.
7. `CODING_RULES.md` и `FILE_INDEX.md` - правила разработки и карта файлов.

Foldora пока является рабочим названием и может измениться перед публичным релизом.
